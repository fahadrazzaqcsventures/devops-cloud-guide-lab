# Runbooks

This document contains **operational runbooks** used during incidents. Runbooks are **action-oriented** and written for use under pressure.

---

## Runbook Template

### Runbook Title:

Clear, searchable title (e.g., "Proxmox Host Network Lockout")

**ID:** RUN-PVE-001

**Owner:** SRE / Infrastructure Team

**Last Updated:** 2025-12-31

**Severity Handled:** Low / Medium / High

### Purpose

What problem this runbook addresses.

### Scope

* Systems affected
* What this runbook does NOT cover

### Preconditions

* Required access (console, SSH, credentials)
* Tools required

### Symptoms

Observable signals:

* Errors
* Alerts
* User impact

### Verification Steps

Commands or checks to confirm the issue.

### Resolution Steps

Step-by-step actions in execution order.

### Validation

How to confirm service restoration.

### Rollback

How to undo changes if resolution fails.

### Escalation

When and how to escalate.

### Post-Incident Tasks

* Logs to collect
* Documentation updates
